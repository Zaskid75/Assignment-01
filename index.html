My Current Target

Web development technologies are the tools and frameworks used to create websites and web applications. They can be broadly categorized into two main parts: front-end and back-end development

HTML

HTML uses tags to mark up different sections and elements of a webpage. These tags tell the web browser how to interpret and display the content, like headings, paragraphs, images, links, and more.

CSS

CSS, which stands for Cascading Style Sheets, is the magic behind the visual appearance of web pages. It's the language that transforms the basic structure of HTML into visually appealing and user-friendly websites.

JavaScript

JS, is a versatile programming language that adds interactivity and dynamism to web pages. It's the third leg of the web development tripod alongside HTML (structure) and CSS (styling).
